Weather App

Description

This Weather App allows users to get current weather information for any city using the OpenWeatherMap API. It features a simple user interface built with HTML, CSS, and JavaScript.

Features

Input a city name to retrieve weather data.
Display temperature, weather description, humidity, and wind speed.
Includes an info button linking to the PM Accelerator’s LinkedIn page.

Tech Stack

Frontend: HTML, CSS, JavaScript
API: OpenWeatherMap API

Getting Started

Prerequisites

A web browser (e.g., Chrome, Firefox)
OpenWeatherMap API key. You can sign up for a free API key

Usage

Enter a city name in the input field.
Click the "Get Weather" button to fetch the weather data.
The weather information will be displayed below the button.
Click the "Info" button to view additional information and a link to PM Accelerator’s LinkedIn page.
